谈到浏览器的网络，就绕不开 HTTP，它是浏览器中应用最多的协议，是浏览器和服务器之间通信的语言，也是互联网的基石

随着浏览器发展，为了适合新的形式也在持续发展。

了解 HTTP 最好的方法就是了解它的发展史。接下来会从以浏览器的发展视角讲讲 HTTP 的研究，分别是

- 即将完成使命的 HTTP/1
- 走向我们的 HTTP/2
- 未来的 HTTP/3
